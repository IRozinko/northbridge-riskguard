# Northbridge RiskGuard

Northbridge RiskGuard is a lightweight B2B risk, responsible gaming and compliance monitoring demo for iGaming operators.

The first MVP module is **RG Monitor**: a responsible gaming and player-risk console that helps risk, compliance and operations teams detect high-risk player behavior, suspicious activity, bonus abuse patterns and review cases before they become regulatory, reputational or financial problems.

## What this MVP includes

- Dashboard with risk and review metrics
- Player risk table
- Basic rule-based risk scoring
- Risk flags:
  - deposit spike
  - loss chasing
  - long session
  - late night activity
  - high deposit frequency
  - failed withdrawals
  - bonus abuse pattern
- Manual review queue
- Events timeline
- Compliance summary / report page
- Planned CSV import screen
- Mock data only

## Important positioning

This project does **not** provide gambling functionality.

It is a validation demo for B2B risk, compliance and responsible gaming workflows.
